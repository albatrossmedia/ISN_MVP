## Writing Guidelines

### Tone and Style

**Professional Yet Approachable**
- Use clear, jargon-free language when possible
- Explain technical concepts with real-world analogies
- Maintain credibility through data and concrete examples
- Show enthusiasm without overhyping

**Google Cloud Partnership Emphasis**
- Always mention Google Cloud partnership in context
- Highlight specific Google Cloud services used
- Emphasize enterprise-grade reliability and security
- Link technical achievements to Google Cloud infrastructure

**Technical Credibility**
- Include specific metrics and performance data
- Reference research papers or technical documentation
- Provide technical depth for expert audiences
- Balance accessibility with technical accuracy

### Content Structure

**Opening (First 2 Paragraphs)**
- Hook readers with the key announcement or story
- Establish relevance and importance
- Mention Google Cloud partnership early when relevant

**Body (Middle Sections)**
- Use subheadings for scannability
- Include bulleted lists for key points
- Add data visualizations and screenshots
- Incorporate quotes from team members or partners

**Conclusion**
- Summarize key takeaways
- Provide clear next steps or call-to-action
- Link to related resources or documentation
- Invite reader engagement (comments, shares, newsletter signup)

## SEO and Metadata Guidelines

### Meta Title Format
- Keep under 60 characters
- Include primary keyword and brand name
- Example: "ISN-ASR v2.1 Released: 95% Accuracy | IndicSubtitleNet"

### Meta Description Format
- Keep under 155 characters
- Summarize key value proposition
- Include call-to-action
- Example: "Discover how IndicSubtitleNet's new AI model achieves 95% accuracy in subtitle generation with Google Cloud. Learn more about our latest innovation."

### URL Slug Best Practices
- Use hyphens to separate words
- Keep under 60 characters
- Include primary keyword
- Example: `/news/isn-asr-v21-release-95-accuracy`

### Image Alt Text
- Describe image content clearly
- Include relevant keywords naturally
- Keep under 125 characters
- Example: "IndicSubtitleNet and Google Cloud partnership milestone achievement ceremony"

4. **Publication:** Override calendar if necessary
